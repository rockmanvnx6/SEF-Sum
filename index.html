Abstract vs Interface

  • An abstract class

    • can have abstract method or not.
    • cannot be instantiated
    • cannot be used with the new
    • but have constructor
    • method can be implemented or overwritten.

  • An abstract method

    • Declare but don’t implement it (act as blue print)
    • cannot be contained in a non-abstract class
    • If the subclass does not implement, It must be declare as abstract.
    • Can exists in interface
